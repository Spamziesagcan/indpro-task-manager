Intern Assignment: Task Manager App

Build a simple Task Manager where users can manage tasks across three stages.

Requirements

Auth

  - Login & Register flow

Tasks

  - Create, update & delete tasks
  - Every task has a stage — Todo, In Progress, Done

UI

  - Clean & responsive design
  - Handle loading & error states
  - How you structure or display it is completely up to you

Tech Stack

Use any framework or language you are comfortable with.

Deployment

  - Frontend: Mandatory (On any free hosting platforms)
  - Backend: Optional & bonus

Backend Note

Backend implementation is optional unless AI tools were used during development.

If you used AI-assisted tools such as Cursor, Copilot, ChatGPT, v0, or similar
tools for any part of the implementation, then backend development becomes
mandatory.

Bonus Features (Optional)

  - Custom backend APIs
  - Database integration
  - Backend authentication (JWT / Sessions)

Submission Requirements

  - GitHub repository with a proper README
  - Mention assumptions, tradeoffs, and technical decisions in the README
  - Frontend live deployment link (Mandatory, On any free hosting platforms)
  - Backend live deployment link (optional - On any free hosting platforms)

Timeline

  - Expected effort: 3-4 hours
  - Submission window: 3–4 days (Till 01st June, 10:00 PM)

We value a small, complete, and well-structured implementation over a large but
incomplete solution.
